# SentinelForge AI

SentinelForge AI is an AI-assisted cybersecurity resilience MVP that demonstrates IOC ingestion, AI-generated threat summaries, workflow health monitoring, supplier cyber resilience scoring, and supply-chain risk analysis.

## Features

- IOC ingestion and classification
- AI-generated IOC threat summaries
- 1min.ai cloud API integration
- Local Ollama model support
- Workflow Health Agent
- External feed monitoring
- Feed latency and status checks
- Supplier profile JSON ingestion
- Supplier resilience scoring
- AI-generated supplier risk summaries

## Tech Stack

- Python
- FastAPI
- Ollama
- 1min.ai API
- Pydantic
- Requests
- python-dotenv
- JSON supplier profiles
